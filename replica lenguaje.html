Algunos lenguajes de la web

HTML +CSS + Javascrip

  • HTML5

    • HTML5 (Hyper Text Markup Language, version 5) es la quinta revision importante de lenguaje basico de la World Wide Web.

    Usa la extension .html

  • CSS3

    • Hoja de estilo en cascada o CSS(siglas en ingles de cascading style sheets)es un lenguaje para definir y crear la presentacion de un documento estructurado escrito en HTML.

    Usa la extension CSS

  • Javascript

    • Javascript (abreviado comunmente "JS")es un lenguaje de programacion interpretado utilizado principalmente en el lado del cliente ( client-side ), implementado como parte de un navegador web permitiendo mejoras enla interfaz del ususario y paginas web dinamicas

    Usa la extencion .js
